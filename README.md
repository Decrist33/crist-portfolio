## Run Locally

Clone the project

```bash
  git clone https://github.com/Decrist33/crist-portfolio.git
```

Go to the project directory

```bash
  cd crist-portfolio
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
